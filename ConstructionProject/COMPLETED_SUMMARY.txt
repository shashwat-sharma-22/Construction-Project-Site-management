TROUBLESHOOTING:

Problem: "View not found"
Solution: Check controller name matches folder name
Example: ProjectController → Views/Project/

Problem: Form doesn't submit
Solution: Ensure form has asp-action attribute
Example: <form asp-action="Create">

Problem: Styling not loading
Solution: Refresh page (Ctrl+F5) or rebuild project

Problem: Database errors
Solution: Run: dotnet ef database update
